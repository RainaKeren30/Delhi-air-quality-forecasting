# Delhi Air Quality Forecasting

An AI-powered air quality forecasting system focused on predicting PM2.5 levels in Delhi using Machine Learning and interactive data visualization.

Built with Python, Streamlit, FastAPI, and modern ML frameworks, this project provides real-time forecasting, environmental insights, and an interactive analytics dashboard.

---

## Features

* PM2.5 air quality prediction
* Interactive Streamlit dashboard
* FastAPI inference API
* Machine Learning forecasting models
* Feature engineering pipeline
* Real-time data visualization
* REST API endpoints
* Forecast analytics and trend monitoring

---

## Tech Stack

* Python
* Streamlit
* FastAPI

* XGBoost

* PyTorch

---

## Run Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Start Streamlit dashboard
streamlit run app/streamlit_app.py

# Run FastAPI server
uvicorn app.main:app --reload
```
